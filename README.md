# RTS 10K Simulation

1만 유닛을 실시간으로 처리하는 대규모 RTS 시뮬레이션 프로젝트입니다.

본 프로젝트는 대규모 유닛 환경에서의 실시간 처리, 시스템 구조, 성능 확장성을 실험하는 것을 목표로 합니다.

---

## 🎯 프로젝트 목표

- 실시간 환경에서 최대 10,000 유닛 동시 처리
- 대규모 에이전트 시뮬레이션 구조 설계
- 성능 병목 분석 및 최적화 실험
- 확장 가능한 RTS 시스템 아키텍처 구축

---

## 🧠 주요 관심사

본 프로젝트는 게임 제작보다는 아래와 같은 시스템적 문제 해결에 초점을 맞추고 있습니다.

- 대규모 유닛 업데이트 구조
- 효율적인 상태 관리
- CPU 부하 최소화
- 시뮬레이션 확장성

---

## ⚙️ 개발 환경

- Unity
- C#
- Built-in Render Pipeline

---

## 🚧 진행 방향

향후 아래와 같은 요소들을 단계적으로 구현 예정입니다.

- 유닛 이동 시스템
- 상태 기반 행동 구조
- 대규모 전투 시뮬레이션
- 성능 측정 및 개선

---
